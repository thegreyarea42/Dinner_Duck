# Dinner Duck 🦆

Dinner Duck is a privacy-focused, shared grocery list and recipe manager built with Flutter. It's designed to simplify your kitchen routine without compromising your personal data.

## Features

### 📅 Meal Planner
Organize your meals by day and type (Breakfast, Lunch, Dinner, Snacks, or Dessert). Scale ingredients on the fly based on the number of servings you need.

### 📚 Digital Cookbook
Save and categorize your favorite recipes. Import recipes directly from websites or export your collection as a backup file.

### 🛒 Intelligent Grocery List
- **Auto-Categorization**: Items are automatically sorted by aisle (Produce, Dairy, Meat, Pantry, etc.) to save you time in the store.
- **Smart Suggestions**: See your most frequently purchased items for quick adding.
- **Staples Management**: Define items you always need and add them all with a single tap.

### 🔊 Accessibility & Usability
- **Voice Instructions**: Hands-busy? Use the built-in text-to-speech to read recipe steps aloud with adjustable speed.
- **Stay Awake**: "Keep Screen On" mode prevents your device from sleeping while you follow a recipe.
- **Customizable Appearance**: Includes a high-contrast Dark Mode (Emerald theme) and adjustable font sizes for easy reading across the kitchen.

### 🔒 Privacy First
- **Local Storage**: All your recipes and lists are stored directly on your device.
- **No Account Required**: Start planning immediately without creating an account or sharing email addresses.

## Getting Started

To run this project, ensure you have the Flutter SDK installed on your machine.

1. **Clone the Repository**:
   ```bash
   git clone https://github.com/thegreyarea42/DinnerDuck.git
   ```

2. **Install Dependencies**:
   ```bash
   flutter pub get
   ```

3. **Launch the App**:
   ```bash
   flutter run
   ```

## Development

Dinner Duck is built using:
- **Flutter & Dart**: For a smooth, cross-platform experience.
- **Shared Preferences**: For fast, local data persistence.
- **Flutter TTS**: For accessible recipe guidance.
