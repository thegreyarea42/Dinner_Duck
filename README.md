# Dinner Duck

A privacy-focused, shared grocery list and recipe manager built with Flutter.

## Features

- **Meal Planner**: Organize your breakfast, lunch, and dinner.
- **Cookbook**: Save your favorite recipes for easy access.
- **Grocery List**: Automatically generate shopping lists from your meal plans.
- **Privacy First**: All data is stored locally on your device.
- **Accessible**: Includes text-to-speech for instructions and adjustable font sizes.

## Getting Started

To run this project, ensure you have the Flutter SDK installed on your machine.

1. Clone the repository.
2. Run `flutter pub get` to install dependencies.
3. Use `flutter run` to launch the application on your connected device or emulator.
